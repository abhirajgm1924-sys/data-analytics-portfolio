# Amazon Price Scraper (Kickoff Project)
This project was one of my first hands-on Python projects while learning data analysis and automation.

## Objective
To collect product price data from Amazon using web scraping and perform basic exploratory analysis.

## Tools Used
Python 
BeautifulSoup  
Pandas  
Matplotlib  

## Workflow
- Scraped product titles and prices from Amazon.
- Stored the collected data in a CSV dataset.
- Cleaned and analyzed the dataset using pandas.
- Visualized price distribution using matplotlib.

## What I Learned
- Basics of web scraping with BeautifulSoup
- Working with datasets using pandas
- Creating visualizations with matplotlib
